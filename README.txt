
# Amina zeigt uns Tiere – Kinderbuch

Dies ist eine einfache Website zum Kinderbuch von Svetlana Noury, bereit für GitHub Pages.

## Anleitung

1. Repository erstellen auf GitHub
2. Diese Dateien hochladen (index.html und Bild)
3. In den Einstellungen GitHub Pages aktivieren
4. URL wird automatisch erstellt: https://<dein-name>.github.io/<repo-name>/
